### 使用说明
面板分为三个部分：普查范围、文件（夹）选择、结果显示。

#### 步骤
1. 在勾选栏中选择你要普查的范围；
2. 输入文件（夹）路径或直接选择文件（夹）；
3. 点击开始普查，即可获得结果。
